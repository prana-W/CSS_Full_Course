As you can see in the page, we are getting an extra horizontal scrollbar in the bottom... which is really annoying.

But why? Because the content is long (vertically) we are getting a scrollbar to the right, which takes some width in our right side of the viewport. The body is set to 100vw, therefore to take the whole 100vw, it will need to increase a bit size to go to that full extend, making that extra horizontal space come and giving us a horizontal scrollbar

FIX: Set the width to 100% instead of 100vw, or none at all!!! If we don't make universally select and make margin back to 0, making width to 100% will still not fix the issue, the best way is to make the default style go back to margin 0, padding 0 and then make use of 100% or don't use the width at all, as we already want 100% which will be the default behaviour
