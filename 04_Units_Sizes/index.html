There are several units for CSS

  • Absolute size unit: pixels (px)- These are absolute and doesn't change size at any cost, as it takes same amount of pixels in all times.
  • Percentage (%)- These are relative to the size of parent element
  • rem - rem unit means "The root element's font-size"
  • em - em unit means "my parent element's font-size" if used for font-size and "my own font-size" when used for anything else
  • ch- ch Character unit. The CSS ch unit is defined as the width of the character 0 (zero, or U+0030) of the font.
  • vw, vh - This is a relative quantity to the width and height of the viewport respectively
