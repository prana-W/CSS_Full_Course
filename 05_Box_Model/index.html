Box Model

We reset the entire page CSS by using universal selector (*). We set margin and padding as 0 and make the box-sizing as border-box.

By setting the box-sizing to border-box, evrything is calculationg including the padding and the border. Example, 400px width of the h1 tag, will be for including the content, width, the padding in both sides and the border in both sides. By default, box-sizing is set to content-box, which only calculates the measurement including the content size only!!
